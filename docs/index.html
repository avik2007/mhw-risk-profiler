SDD Label

Stress Degree Days [°C·day] — cumulative SST above the Hobday MHW threshold. Computed from HYCOM water_temp[depth=0] against a 30-year OISST v2.1 baseline.

Ensemble Design

ERA5: single reanalysis expanded to 64 synthetic members via Gaussian perturbation. WN2: genuine 64-member physical ensemble from Google DeepMind's WeatherNext 2.

GCS Infrastructure

Multi-year Zarr tiles cached in GCS for fast Dask-backed training. Crash-safe per-month idempotent writes with .complete sentinels.

Model Architecture

1D-CNN + Transformer + LeakyGate

Per-cell spatiotemporal encoder with atmosphere–ocean attention gating.

1D-CNN Depth Encoder — compresses HYCOM's 10-level depth profile into a fixed-size ocean state vector per grid cell.
Transformer Temporal Encoder — captures 90-day seasonal context in both atmospheric and oceanic streams.
LeakyGate (α) — learned scalar per cell blending atmospheric and oceanic representations. ERA5 converges to α ≈ 0.26; WN2 to α ≈ 0.44. The difference reflects each ensemble's ability to resolve atmosphere–ocean coupling.
Gaussian Spread Head — predicts mean SDD and spread across the 64-member ensemble. Spread quantifies intra-ensemble uncertainty.
SVaR Aggregation — sorts per-member SDD predictions to produce the 5th, 50th, and 95th percentile Stress Value-at-Risk across cells.
Loss: MSE on normalized SDD (÷ 250 °C·day) · Adam · lr=1e-4 · early stop patience=10

Roadmap

v1 → v2 Improvements

v1 establishes the pipeline and architecture. v2 addresses the core scientific limitations.

v1 — Current
2-year OISST baseline (requires ≥30yr for Hobday 2016 threshold)
HYCOM labels deterministic across all 64 members — spread underestimated
Partial SVaR inference (5 of 161 cells) — full run requires GCS
Model trained on 2 years; limited temporal generalization
v2 — In Progress
30-year OISST v2.1 climatology (1982–2011) for calibrated MHW thresholds
GLORYS12V1 ocean reanalysis + AR(1) perturbation → stochastic labels
Full spatial SVaR inference across all 161 ocean cells
Extended to 2022–2025 using GLORYS12V1 coverage
